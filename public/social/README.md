# Kit logo/social ConText

Questa cartella contiene asset pronti per aprire e presentare la pagina TikTok di ConText.

| File | Uso consigliato |
|---|---|
| `logo-tiktok-profile-1080.png` | Immagine profilo TikTok o social in formato quadrato grande. |
| `logo-square-512.png` | Logo quadrato compatto per miniature, avatar secondari o app. |
| `logo-transparent-wordmark.png` | Logo con scritta su sfondo trasparente, utile sopra copertine o grafiche. |
| `tiktok-cover-1920x1080.png` | Cover orizzontale 16:9 per video, anteprime o materiali promozionali. |

Gli asset sono generati dagli elementi grafici già presenti nella PWA e mantengono palette scura, oro e teal di ConText.
